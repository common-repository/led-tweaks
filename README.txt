A suite of common customizations, shortcodes and tweaks for Wordpress and Themeco's Pro theme as used by Liquid Edge.

== Installation ==

1. Upload `led-tweaks` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
